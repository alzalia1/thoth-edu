< retour
Enregistre les réponses d'un élève

Status

⚠️ En test

Requête

                        
{
    "id_el" : "pseudo de l'élève",
    "id_access" : "id de l'accès auquel l'élève répond",
    "answers" : [
        {
            "id" : "id de la question associée",
            "answer" : "réponse de l'élève"
        }, 
        ...
    ]
}
                        
                    
                        
{
    "id_el" : "pseudo de l'élève",
    "id_access" : "id de l'accès auquel l'élève répond",
    "answers" : [
        {
            "id" : "id de la question associée",
            "answer" : [["", ... ], ... ]
        }, 
        ...
    ]
}

Réponse

                    
{
    "status" : "fail ou success",
    "reason" : "raison du fail ou none"
}
Remarque : reason peut soit contenir "none", soit ne pas être renvoyé du tout dans le cas d'un succès.

Exemple

                            
{
    "id_el" : "Jeremy N.",
    "id_access" : "w02fuh9d",
    "answers" : [
        {
            "id" : "1",
            "answer" : "mesa"
        }, 
    ]
}
                            
                        
                            
{
    "id_el" : "Jeanne D.",
    "id_access" : "7a6spqds",
    "answers" : [
        {
            "id" : "1",
            "answer" : [["mange", "mangerai"], ["manges", "mangeras"], ["mange", "mangera"]]
        },
    ]
}
→
                            
{
    "status" : "success",
}
Remarque : On a fait ici le choix de ne pas mettre reason vu que le status est "success".
