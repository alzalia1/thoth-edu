Documentation des API du site thoth-edu

https://api.thoth-edu.fr/
├── user/
│   ├── ✅ check - Vérifie la validité d'un token JWT et le rafraîchit
│   ├── ✅ login - Connecte un utilisateur et lui donne un token JWT
│   ├── ✅ signup - Inscris un utilisateur dans la base de données
│   ├── 🔨|📡 delete - Supprime un utilisateur de la base de données
│   └── 📡|❓ logout - Déconnecte un utilisateur en invalidant son token JWT
├── dashboard/
│   ├── eval/
│   │   ├── ✅ get - Obtiens les informations relatives à une eval pour modification
│   │   ├── ✅ create_access - Ajoute un accès à la base de données
│   │   └── ✅ delete - Supprime l'évaluation
│   ├── access/
│   │   ├── 🔨|📡 get - Obtiens les informations relatives à un accès
│   │   ├── 🔨|📡 delete - Supprime un accès de la base de données
│   │   └── 📡|❓ save - Enregistre des modifications apportées à l'accès
│   ├── ans/
│   │   ├── 📡|❓ get - Obtiens les informations relatives à la copie d'un élève
│   │   └── 📡|❓ delete - Supprime la copie d'un élève
│   └── ✅ infos_user - Obtiens les informations relatives à un utilisateur
├── crea/
│   └── ✅ save - Ajoute une évaluation à la base de données
└── eval/
    ├── ✅ check_access - Vérifie l'existence d'un accès par son ID
    ├── ✅ get_eval - Obtiens les informations relatives à une évaluation pour élève
    └── ⚠️ ans_student - Enregistre les réponses d'un élève dans la base de données
